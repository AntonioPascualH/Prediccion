Leave-One-Out Cross-Validation

El LOOCV consiste en tomar una muestra con todos los datos menos uno. Se estima el modelo y se predice sobre el dato que se ha dejado fuera. Este proceso se repite para los n datos.

K-Fold Cross-Validation

Supone dividir la muestra en k grupos o folds, de aproximadamente igual tamao. Cada folds es tratado como un conjunto de validacin, de tal forma que se estima el modelo con los datos que no estn el fold (los otros k1 folds) y se predicen en el fold.
